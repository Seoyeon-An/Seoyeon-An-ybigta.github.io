2023 YBIGTA DA Junior Session

연세대학교 빅데이터 학회 YBIGTA는 2012년 창립된 국내 최초 빅데이터 학술 학회로,

요즘 시대의 가장 핫한 키워드인 '빅데이터'를 직접 다루고 분석하는 스터디 및 프로젝트 역량을 발전시킬 수 있는 학회입니다.

YBIGTA의 DA(Data Analytics) 팀은 데이터 기반의 의사결정을 위해 문제 정의 및 데이터 수집부터 전처리, 시각화까지 데이터 분석의 All-Rounder를 지향하는 팀입니다.

Session 1. SQL

SQL이란 Structured Query Language의 줄임말로, RDBMS에서 자료를 관리, 처리하기 위해 설계된 언어를 말합니다.
데이터 분석가의 가장 핵심적인 skill 중 하나가 직접 데이터베이스에 접근하여 분석에 필요한 데이터를 SQL로 추출할 수 있는가에 대한 것인 만큼,
DA 분야에 관심이 있다면 SQL에 대해 공부하고 연습하는 것은 필수적인 과정이라고 이야기할 수 있습니다.

SQL은 두 주차로 나누어 세션이 진행되었는데, 첫 번째 세션에서는 기본적인 SQL의 문법들을 익혀보았습니다.
SELECT FROM, WHERE, GROUP BY 등 기본적인 내용부터 JOIN의 종류, Subquery에 대한 내용들도 배울 수 있었습니다.
또한, 보다 advanced한 내용으로 JOIN vs Subqery, Optimizer Hint 등과 같은 SQL 최적화에 관한 내용들도 다루는 시간을 가졌습니다.

Session 2. SQL 활용

두 번째로 진행된 SQL 활용 세션에서는 개념적으로 SQL에 접근해보는 시간을 가졌습니다.
RDBMS, DDL, Data Type 등에 대한 개념들과, Schema 디자인이 어떻게 이루어지는지, 정규화 및 관계차수, ER Diagram 등
보다 체계적으로 RDBMS로서의 SQL을 차근차근 공부해볼 수 있는 시간이었습니다.

이렇게 개념을 익힌 이후, 직접 실습 데이터 및 파일을 이용하여 MySQL DB를 구축하고,
이를 활용하여 배웠던 Query문들을 적용해보는 실습 시간을 가졌습니다.
